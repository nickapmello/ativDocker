Bem-vindo ao Meu Site com Docker!

Este site está rodando em um contêiner Docker com:

  • Nginx como servidor web para servir esta página.
  • MySQL como banco de dados para armazenar informações.

Docker facilita a configuração e o gerenciamento de ambientes isolados, permitindo um desenvolvimento mais eficiente e consistente.
